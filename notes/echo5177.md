\- 链上产品和普通互联网产品有什么不同：普通互联网产品通常由公司服务器保存用户账号、数据和业务状态，用户用账号密码登录，操作结果主要记录在平台数据库里。链上产品则把关键状态和交易记录写到区块链上，用户通过钱包签名确认操作，地址和交易可以在区块浏览器公开查询。因此链上产品更透明、可验证，也更强调用户自己保管资产和私钥安全。

1\. Testnet 是什么？

\---------------

**Testnet = 测试网。**

第一性原理地说，区块链网络本质上是一群节点按照同一套规则维护同一本账本。\*\*Mainnet 主网\*\*是真钱、真实资产、真实交易；\*\*Testnet 测试网\*\*是给开发者和学习者练习用的“仿真环境”。Ethereum 官方把 testnet 类比为软件开发里的 **staging server / 预发布环境**：在部署到主网前，先在测试网测试合约和应用。测试网资产一般没有真实价值，通常通过 faucet 免费领取。(\[[ethereum.org](http://ethereum.org)\]([https://ethereum.org/developers/docs/networks/](https://ethereum.org/developers/docs/networks/) "Networks | [ethereum.org](http://ethereum.org)"))

你拿到的 **55 MON** 是 Monad Testnet 上的测试 MON，作用是：

支付测试网 gas

测试转账

测试合约交互

测试 DApp

它不是主网 MON，也不是可当真钱使用的资产。

2\. 当你在 MetaMask 里选择不同 Networks 时，发生了什么？

\---------------------------------------

你不是在“换钱包”，而是在让同一个钱包连接到\*\*不同的区块链账本\*\*。

可以这样理解：

你的钱包地址：

0xABC...

在 Ethereum Mainnet 上：

可能有 0.1 ETH

在 Monad Testnet 上：

可能有 55 MON

在 Sepolia 上：

可能有 0.5 Sepolia ETH

在 Polygon 上：

可能有 10 MATIC

地址看起来可以是同一个，但它连接的是不同网络，所以余额、交易历史、Gas 代币、区块浏览器都不同。Ethereum 官方文档也说明，同一个 Ethereum 账户可以在不同网络上使用，但主网上的余额和交易历史不会自动带到其他网络。(\[[ethereum.org](http://ethereum.org)\]([https://ethereum.org/developers/docs/networks/](https://ethereum.org/developers/docs/networks/) "Networks | [ethereum.org](http://ethereum.org)"))

所以你切换 Network，本质上是在切换：

RPC：钱包向哪个网络节点询问数据

Chain ID：当前是哪条链

Gas Token：用什么币付手续费

Explorer：去哪里查交易

State：读取哪一本账本

Monad Testnet 的 Chain ID 是 **10143**，Token 是 **MON**，官方开发者入口也列出了 RPC、Explorer 和 Faucet 等信息。(\[Monad Developer Portal\]([https://developers.monad.xyz/](https://developers.monad.xyz/) "Monad Developer Portal"))

为什么有这么多网络？因为不同链在做不同事情：

Ethereum Mainnet：真实生产环境

Sepolia / Hoodi：Ethereum 测试环境

Monad Testnet：Monad 的测试环境

Polygon / Arbitrum / Base：不同扩容链或生态链

本地网络：开发者自己电脑上跑的临时链

Ethereum 官方也说，除了唯一的 Ethereum Mainnet，任何人都可以创建遵循相同协议规则但彼此不交互的独立网络，用于测试和开发。(\[[ethereum.org](http://ethereum.org)\]([https://ethereum.org/developers/docs/networks/](https://ethereum.org/developers/docs/networks/) "Networks | [ethereum.org](http://ethereum.org)"))

3\. 区块浏览器是什么？

\------------

**区块浏览器 = 区块链的公开查询网站。**

普通互联网里，你的订单、支付记录、后台日志在公司数据库里，外人看不到。链上世界里，交易、地址、区块、合约调用是公开账本的一部分，所以需要一个“搜索引擎”把这些数据展示出来。

区块浏览器可以查：

地址 Address

余额 Balance

交易 Transaction

交易哈希 Transaction Hash

区块 Block

合约 Contract

Gas Fee

交易成功或失败

Etherscan 自称是 Ethereum 的区块浏览器和分析平台，可以探索交易、地址、代币、价格和其他链上活动；Monad 也有自己的测试网浏览器，比如 Monadscan / MonadVision。(\[Ethereum (ETH) Blockchain Explorer\]([https://etherscan.io/index?utm\_source=chatgpt.com](https://etherscan.io/index?utm_source=chatgpt.com) "Ethereum (ETH) Blockchain Explorer - Etherscan"))

一句话：

\> **钱包负责发起操作，区块浏览器负责验证这件事是否真的发生在链上。**
