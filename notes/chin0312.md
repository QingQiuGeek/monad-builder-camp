### Blockchain

-   去中心化分布式账本（Distributed Ledger）
    
-   Block = 交易数据 + 上一区块 Hash
    
-   Chain = 多个 Block 按时间连接
    

### 特性

-   去中心化（Distributed）
    
-   不可篡改（Immutable）
    
-   公开透明（Transparent）
    
-   匿名（Address ≠ Identity）
    

### 运作流程

User  
→ 发起 Transaction  
→ Broadcast  
→ Validator 验证  
→ 打包 Block  
→ 上链  
→ Validator 获得 Reward + Gas Fee

### 为什么需要 BTC？

-   激励节点维护网络
    
-   支付网络奖励
    

### 三种链

Public Chain

-   人人可加入
    
-   去中心化
    
-   BTC、Ethereum
    

Consortium Chain

-   多机构共同管理
    
-   半开放
    

Private Chain

-   企业内部
    
-   中心化
    

### Web2 vs Web3

Web2

-   Platform owns data
    
-   Trust Platform
    

Web3

-   User owns data/assets
    
-   Wallet + Smart Contract
    
-   Trust Code
    

### Web3 核心

Wallet  
→ Sign  
→ Smart Contract  
→ Blockchain

### 去中心化优点

-   Trustless
    
-   Anti-censorship
    
-   User Ownership
    
-   Open Innovation
    

### 缺点

-   TPS 较低
    
-   Gas 高
    
-   UX 差
    
-   法规不成熟
