## Web3 合规（Compliance）

### 中国监管基调

-   禁 ICO / IEO / IDO
    
-   禁虚拟币交易所服务中国大陆用户
    
-   禁挖矿
    
-   虚拟币不是法币
    
-   Web3 技术合法，但金融活动监管严格
    

### 高风险行为

-   发 Token 融资
    
-   赌博（充值→抽奖→提现）
    
-   多级返佣（传销）
    
-   OTC 非法换汇
    
-   洗钱
    
-   非法集资
    

### Web3 工作注意

-   确认公司是否合法
    
-   看雇佣主体（是否有劳动合同/EOR）
    
-   留意社保、公积金
    
-   Token 薪资有价格波动及合规风险
    
-   出金尽量走正规渠道
    

## 全球监管

重点

-   KYC（Know Your Customer）
    
-   AML（Anti-Money Laundering）
    
-   FATF（全球反洗钱组织）
    
-   Travel Rule（大额转账需传递双方信息）

## 网络安全（Security）

**助记词 = 钱包本身。**

任何人要助记词 = 骗子。

## 最常见攻击

### Phishing（钓鱼）

伪装：

-   官网
    
-   HR
    
-   Discord
    
-   Telegram
    
-   空投
    
-   面试
    

目的：

-   骗你连接钱包
    
-   骗你签名
    
-   骗助记词
    

### Malware（木马）

来源：

-   面试软件
    
-   破解软件
    
-   ZIP
    
-   PDF
    
-   Word
    
-   浏览器插件
    

可能造成：

-   偷 Cookie
    
-   偷密码
    
-   偷钱包
    
-   远程控制电脑
    

### Social Engineering（社工）

冒充：

-   HR
    
-   官方
    
-   学长
    
-   朋友
    

利用信任骗你操作。

### Clipboard Attack（剪贴板劫持）

复制钱包地址

↓

自动替换成黑客地址

所以：

**每次转账必须核对完整地址。**

### Browser Extension

只安装：

-   MetaMask
    
-   Rabby
    
-   Bitwarden
    
-   uBlock Origin
    

不要乱装插件。

### Blockchain

-   去中心化分布式账本（Distributed Ledger）
    
-   Block = 交易数据 + 上一区块 Hash
    
-   Chain = 多个 Block 按时间连接
    

### 特性

-   去中心化（Distributed）
    
-   不可篡改（Immutable）
    
-   公开透明（Transparent）
    
-   匿名（Address ≠ Identity）
    

### 运作流程

User  
→ 发起 Transaction  
→ Broadcast  
→ Validator 验证  
→ 打包 Block  
→ 上链  
→ Validator 获得 Reward + Gas Fee

### 为什么需要 BTC？

-   激励节点维护网络
    
-   支付网络奖励
    

### 三种链

Public Chain

-   人人可加入
    
-   去中心化
    
-   BTC、Ethereum
    

Consortium Chain

-   多机构共同管理
    
-   半开放
    

Private Chain

-   企业内部
    
-   中心化
    

### Web2 vs Web3

Web2

-   Platform owns data
    
-   Trust Platform
    

Web3

-   User owns data/assets
    
-   Wallet + Smart Contract
    
-   Trust Code
    

### Web3 核心

Wallet  
→ Sign  
→ Smart Contract  
→ Blockchain

### 去中心化优点

-   Trustless
    
-   Anti-censorship
    
-   User Ownership
    
-   Open Innovation
    

### 缺点

-   TPS 较低
    
-   Gas 高
    
-   UX 差
    
-   法规不成熟
