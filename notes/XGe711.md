# 2026-07-08
<!-- DAILY_CHECKIN_2026-07-08_START -->



今天学习了 AI Agent 如何拥有支付能力  
  
AI Agent 的支付能力可以分为四类：

1.  **调用传统支付 API**：适合企业应用，Agent 代替用户完成支付。
    
2.  **Web3 钱包支付**：Agent 持有链上钱包，自主发起加密资产交易，是当前最受关注的模式。
    
3.  **通过 MCP 等工具协议调用支付服务**：将支付能力封装为工具，便于集成和权限管理。
    
4.  **自主经济体（Autonomous Economic Agent）**：Agent 能赚钱、持有资产、消费和参与链上经济，但目前仍处于快速发展阶段。
