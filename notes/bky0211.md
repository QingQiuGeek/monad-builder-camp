## 五、解释这笔交易

```Plain
Transaction Hash：交易哈希，交易的小票编号
Status：交易状态，成功或失败
Block：这笔交易被打包进哪个区块
Timestamp：发生时间
From：发送方地址
To：接收方地址或合约地址
Value / Amount：转了多少 MON
Gas Fee：这笔交易花了多少手续费
Input Data：如果是合约交互，会显示调用数据
```
